👋 Olá e bem-vindo ao teu primeiro mergulho na Web!

Este tutorial foi criado especialmente para ti, participante do Verão ULisboa na FCUL 🌞

Aqui vais aprender a criar o teu próprio site com HTML e CSS, mesmo que nunca tenhas programado antes!

💡 HTML é a linguagem usada para estruturar páginas na internet.

🎨 CSS é o que usamos para dar estilo e tornar a página bonita.

Neste tutorial vais:

  • 👨‍💻 Escrever código HTML e CSS
  • 🧪 Ver resultados em tempo real
  • 🎯 Receber pontos por cada passo concluído
  • 🏅 Obter um certificado final com o teu nome

⚙️ Vamos começar por escrever o nosso primeiro título com HTML!

Pensa no título como o cabeçalho de um jornal — importante e destacado!

💡 Sabias que todos os sites, incluindo Google, YouTube e FCUL, usam HTML?

Passo 1: Cria um título com <h1>

O HTML usa "tags" para dizer ao navegador o que mostrar. Um título é escrito assim:

<h1>A minha primeira página</h1>

📝 Escreve esse código no editor abaixo. Depois clica em "Atualizar Preview" para ver o resultado.

🔁 Importante: Altera o texto dentro do <h1> para algo pessoal, como o nome da tua receita favorita, o teu animal preferido, ou outra ideia criativa!

🏅 Vais ganhar pontos por cada passo concluído corretamente.

⏱️ O tempo também está a ser contado — será mostrado no teu certificado final!

Experimenta mudar o texto entre as tags <h1> e </h1>. Ex: <h1>Pizza Margherita</h1>

Passo 2: Adiciona um parágrafo com <p>

Os parágrafos em HTML são criados com a tag <p>. São ótimos para escrever texto normal, como este que estás a ler!

Exemplo:

<p>Este é o meu primeiro parágrafo!</p>

📝 Escreve um parágrafo no editor e vê o resultado clicando em "Atualizar Preview".

Certifica-te de que escreveste o parágrafo entre as tags <body> e </body>.

Passo 3: Muda a cor de fundo do site usando CSS com background-color.

Podes escolher uma cor clicando abaixo e copiar o código gerado para usares no editor.

Usa a cor dentro da tag <style> no <head>:
<style>
  body { background-color: #xxxxxx; }
</style>
💡 Sabias que também podes usar nomes como lightblue, lavender ou beige?

Passo 4: Insere uma imagem com <img>

Usa a tag <img> com os atributos src e alt para adicionar uma imagem à tua página.

<img src="https://ciencias.ulisboa.pt/sites/default/files/Ciencias_Logo_Azul-02.png" alt="FCUL Logo">

📝 Copia o exemplo (alterando a URL e o texto do alt) para o editor e vê no preview.

🔁 Podes usar qualquer imagem da internet — só precisas do link direto para a imagem! (Procura a imagem da tua refeição preferida)

Verifica se o atributo alt descreve bem a imagem — é importante para acessibilidade!

Passo 5: Adiciona um hiperligação com <a>

Para criar links, utiliza a tag <a> com o atributo href:

<a href="https://www.ulisboa.pt" target="_blank">Visita a ULisboa</a>

Procura o site da FCUL na net e adiciona essa hiperligação.

O atributo target="_blank" faz o link abrir numa nova aba.

Passo 6: Cria listas com <ul> e <ol>

As listas não ordenadas usam <ul>, e as ordenadas usam <ol>:

Introduz uma lista sobre as coisas que já fizeste hoje.

<ul>
          <li>{Item A}</li>
          <li>Item B</li>
        </ul>

Podes aninhar listas dentro de outras para sub-itens.

Passo 7: Usa <strong> e <em>

Aprende a enfatizar texto com tags semânticas:

<p>Este texto é <strong>importante</strong> e este é <em>itálico</em>.</p>

<strong> indica ênfase forte; <em> indica ênfase normal.

Passo 8: Cria e aplica uma classe CSS

No teu <style>, define uma classe:

.highlight {
          background-color: yellow;
          padding: 4px;
        }

Depois, no HTML:

<p class="highlight">Este texto fica realçado.</p>

Classes permitem reutilizar estilos em vários elementos.

Passo 9: Estiliza um elemento por ID

Em style.css:

#meu-titulo {
          color: darkblue;
          font-size: 32px;
        }

No HTML:

<h2 id="meu-titulo">Título Especial</h2>

IDs são únicos — só podes usar uma vez por página.

Passo 10: Organiza conteúdo com Flexbox

Cria um container flexível:

.container {
          display: flex;
          gap: 16px;
        }

No HTML:

<div class="container">
          <div>Box 1</div>
          <div>Box 2</div>
        </div>

Experimenta justify-content e align-items.

Passo 11: Cria uma tabela com <table>

<table>
          <tr><th>Nome</th><th>Idade</th></tr>
          <tr><td>{O teu nome}</td><td>{A tua idade}</td></tr>
        </table>

Usa border em CSS para linhas visíveis.

Passo 12: Adiciona um formulário

<form>
          <label>Nome: <input type="text"></label>
          <button type="submit">Enviar</button>
        </form>

Lembra-te de usar for no <label> para acessibilidade.

Passo 13: Estiliza links ao passar o rato

a:hover {
          color: red;
          text-decoration: underline;
        }

📝 Adiciona um link e testa o efeito no hover.

Podes usar pseudo-classes também em botões e divs.

Passo 14: Ajusta espaçamentos

.caixa {
          margin: 20px;
          padding: 10px;
          border: 1px solid #333;
        }

Margin é fora da caixa; padding é por dentro.

Passo 15: Define fonte e tamanho

p {
          font-family: 'Inter', sans-serif;
          font-size: 18px;
          line-height: 1.5;
        }

Line-height melhora a leitura em blocos de texto.

Passo 16: Faz as imagens ajustarem-se

img {
          max-width: 100%;
          height: auto;
        }

Isto impede que a imagem “escape” do container.

Passo 17: Responsividade com @media

@media (max-width: 600px) {
          body {
            background-color: lightgray;
          }
        }

Testa redimensionando a janela do browser.

Passo 18: Organiza com CSS Grid

.grid {
          display: grid;
          grid-template-columns: repeat(3, 1fr);
          gap: 10px;
        }

O 1fr divide o espaço igualmente.

Passo 19: Usa custom properties

:root {
          --main-color: teal;
        }
        h1 {
          color: var(--main-color);
        }

Variáveis facilitam manter um tema consistente.

Passo 20: Adiciona uma transição

button {
          background-color: blue;
          transition: background-color 0.3s;
        }
        button:hover {
          background-color: darkblue;
        }

Podes também criar animações com @keyframes.

Passo 22: Cria uma secção “Sobre Mim”

Agora é a tua vez de te apresentares ao mundo! ✨

📝 Cria uma secção no teu site onde falas um pouco sobre ti. Podes escrever o que quiseres — este espaço é teu!

Algumas ideias para começares:

  • O teu nome, idade ou cidade
  • O que gostas de fazer nos tempos livres
  • O teu animal, comida ou jogo favorito
  • Algo engraçado ou único sobre ti

💡 Usa tudo o que aprendeste até agora: títulos, parágrafos, listas, imagens, estilos com CSS… até emojis se quiseres!

🌟 Este é o momento em que podes personalizar o teu site à tua maneira. Podes alterar, apagar ou reorganizar o que quiseres no código — o importante é que o resultado final seja a tua cara! 🎨

🙋‍♀️ Se tiveres dúvidas ou precisares de inspiração, chama um monitor — eles estão aqui para te ajudar.

Experimenta usar tags como <h2>, <p>, <ul> e <img> com uma imagem tua ou de algo que gostes.
